Pineapple Music App

API Endpoints

GET/api/genre

Gets all the genres

GET/api/instrument

Gets all the instruments

GET/api/profile/:type/:id

Get band or musician profile by id

:type is band or musician
:id is band or musician object id

GET/api/connection/:type/:id

Get connections lists by ids

:type is heaven, hell or theirheaven
:id is band or musician source_id
- heaven - lists heaven (favourites)
- hell - list hell (blocked)
- theirheaven - lists who favourited current user
