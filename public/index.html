Pineapple Music App

API Endpoints

GET/api/genre

Get all the genres

GET/api/instrument

Get all the instruments

GET/api/match/band/:id

Get matches band matches by musician id

GET/api/match/musician/:id

Get matches musician matches by band id

GET/api/profile/band/:id

Get band profile by user id

GET/api/profile/musician/:id

Get musician profile by user id

GET/api/ilike/:id

Get user liked list by user id

GET/api/likesme/:id

Get list of likes by user id

POST/api/profile/musician

Create new musician profile

POST/api/profile/band

Create new band profile

POST/api/profile/band/:id

Updates band profile by user id

POST/api/profile/musician/:id

Updates band profile by musician id

POST/api/connections

Creates connection entry

POST/api/donotdisplay

Creates DND entry
